저는 한양사이버대학교 컴퓨터공학부 4학년입니다. 졸업 프로젝트로 학습 보조 웹 어플리케이션을 개발하고 있어요. 카드를 작성하고 일정 기간마다 자동으로 복습할 수 있도록 도와주는 웹어플리케이션입니다.

🎯 프로젝트 개요
사용자가 카드를 만들고, 일정 기간 반복적으로 학습할 수 있는 스마트 학습 관리 웹 서비스를 만들고 있습니다.

🛠 구현 기능과 기술 스택
Frontend : React + TailwindCSS
Backend : NodeJS + React router
Database : SQLite + Prisma
UI Styling : TailwindCSS + Lucide Icons
